# NEUSSA Website

A modern, responsive site for the Northeastern University Somali Student Association, built with React, TypeScript, and Vite.

**Live Site:** [northeasternsomalistudentassocation.org](https://northeasternsomalistudentassocation.org)

---

## Tech Stack
- React 18
- TypeScript
- Vite
- React Router
- Custom CSS
- Deployed on GitHub Pages (custom domain)

---

## Quick Start
```bash
git clone https://github.com/M2theI/SSA-Website.git
cd SSA-Website
npm install
npm run dev
```
Visit [http://localhost:5173/](http://localhost:5173/) in your browser.
